## Make

```
make ydkb/unicore_f1:xxx_vial
```

xxx means taco75 or kbd8xmk3 or tacopad ...

## Convert BIN to UF2

```
uf2conv.py ydkb_unicore_f1_xxx_vial.bin -b 0x8004000 -c -f 0x9d5bcf10 -o xxx.uf2
```
